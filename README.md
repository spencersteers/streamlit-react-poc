# streamlit-react-poc

Example streamlit + react setup.

## Setup

Both streamlit and create-react-app need to be running.

### Streamlit
```
streamlit run app.py
```

### create-react-app
```
cd streamlit_custom_input/frontend
npm install
npm run start
```
